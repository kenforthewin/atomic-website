The Atomic iOS app is a native SwiftUI client that connects to your `atomic-server` instance.

## Requirements

- iOS 17+
- A running `atomic-server` instance accessible from your device
- An API token for authentication

## Setup

1. Install the app on your iOS device
2. Enter your server URL (e.g., `https://atomic.yourdomain.com`)
3. Enter your API token
4. The app will connect and sync your atoms

## Features

The iOS app focuses on reading and writing atoms on the go:

- Browse and search your atoms
- Create new atoms with markdown
- View tags and filter by tag
- Full-text search

The iOS app is a thin HTTP client — all data lives on your server.
